# Sales ETL Pipeline & BI Dashboard Data

> **Type:** ETL Pipeline  
> **Author:** Prajwal Markal Puttaswamy  
> **Stack:** Python · Pandas · SQLite · Matplotlib

## Overview

Automated Python ETL that ingests raw sales data, cleans it, computes KPIs, and outputs dashboard-ready datasets.

## Tech Stack

- Python
- Pandas
- SQLite
- Matplotlib

## Getting Started

```bash
# Clone the repo
git clone https://github.com/prajwal467262/sales-etl-pipeline-&-bi-dashboard-data

# Install dependencies
pip install -r requirements.txt

# (If applicable) Generate sample data first
python generate_sample_data.py

# Run the main script
python sales_etl_pipeline.py
```
